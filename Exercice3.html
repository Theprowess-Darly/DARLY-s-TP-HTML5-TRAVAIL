CSS pour personnaliser davantage le style de vos paragraphes

color: red;: Cette propriété définit la couleur
du texte de ces paragraphes en rouge. Vous pouvez
remplacer "red" par n'importe quelle autre couleur
(nom de couleur, code hexadécimal, fonction RGB, etc)

Lorsque vous ouvrirez ce code HTML dans un navigateur
web, vous verrez une liste ordonnée avec trois éléments, chacun
étant numéroté et affiché en bleu avec une police plus grande.
